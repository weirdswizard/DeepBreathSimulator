#  Deep Breath Simulator

A Roblox simulator where players dive into the ocean, explore deeper zones, and earn rewards while managing breath, upgrades, pets, and rebirth progression.

##  Features

*  Diving system with depth tracking
*  Breath system with upgrades
*  Coin rewards based on depth
*  Pet system with multipliers
*  Rebirth system for progression
*  Zone-based ocean exploration

##  Tech Stack

* Roblox Studio
* Rojo (project syncing)
* Git + GitHub (version control)

##  Status

Core systems completed. Currently improving UI, progression, and polish.

##  Future Plans

* Gamepasses & monetization
* Better UI
* More zones and pets
